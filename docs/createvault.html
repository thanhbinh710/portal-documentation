Create a new Vault

Follow these steps to create your first vault

  1. Navigate to Publisher View from the header
  2. Click on CREATE on manage vault screen
  3. Enter your vault name (e.g. Testing Vault)
  4. Select a team to deploy your app (e.g. guest) and the Authorization Type (Inherit auth from parent)
  5. Fill up at least one parameter for your vault. Supposed that you want to add a "x-subscription-key" field in the Header parameter, you can click on the Add field* button and fill up the key-value fields accordingly.
key: "x-subscription-key"
value: "YOUR_SUBSCRIPTION_KEY_HERE"
  1. Click on CREATE button

At the end of creation, you can see your new vault on manage vault screen
