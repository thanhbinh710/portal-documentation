Using the database provided

Create a new data class

Supposed that you want to create a new data class called Pet, you can follow these steps below

  1. Navigate to Database > Users
  2. Click on Create a class
  3. Key in the class name (e.g. Pet)
  4. Click on Create to add

Create a new column in the class

Supposed that you want to create a new column called Name in Pet, you can follow these steps below

  1. Click on Add a new column on the top right hand corner
  2. Key in the name of the column (e.g. name)
  3. Click on Add column to add

Create a new row in the class

  1. Click on Add a row in the middle of the page
  2. Key in the name of the pet you want to add (e.g. Dog). Noted that the default fields such as Object Id, updatedAt, createdAt, ACL are automatically created.
  3. Press enter to create

Retrieving the created data in Postman

Upon success creation, you can make an API call in Postman to retrieve the data that you just created. Open the ACNAPI Common Service Collection Postman collection, select the Get Data Object endpoint (under Data folder)

  • Copy the Server Token from Authentication (Reference)
  • Change the api to https://ug-api-dev.acnapiv3.io/common/classes/Pet
  • Press Send
