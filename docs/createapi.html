Upload a new API Collection

Follow these steps to add a new API Collection

  1. Navigate to Manage API from the sidebar
  2. Click on CREATE on manage API screen
  3. Enter your API name, API Hostname
  4. Select whether you want to protect your API with a Vault
  5. Select you want the public to have access to your API
  6. Upload your API Postman Collection by clicking on the Upload File button
  7. Click on SAVE AND CONTINUE to go to the next step
  8. Provide a short description for the API and select a category
  9. Clicked SAVE AND CONTINUE to submit your API for review

You will then see the status of your API as Pending on the manage API screen
