Getting Started

acnapi Portal aims at creating reusable building blocks as APIs for easy repurposing cross-industry, enabling you to deliver value through SaaS models.

What is it used for​?

1. ASSETS CREATION​

  • Reusable building blocks in APIs and templates​

  • Multi-tenancy​

  • Full-stack automation + DevOps​

  • Governance enforcement​

2. EXPERIMENTATION

  • Assembly approach enabling rapid prototyping and MVP/pilot​

  • Built-in analytics enabling measure, learn and iterate​

  • Internal and external (startups) building blocks for co-innovation​

3. PRODUCTION

  • Cloud-first architecture fully leveraging public clouds​

  • Cloud agnostics (AWS, GCP, Azure)​

  • API Analytics​

  • Security​

  • Monitoring and automated functional testing​
