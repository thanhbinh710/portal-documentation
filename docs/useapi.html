Using a subscribed API

Open Postman Documentation

Supposed that you want to use the APIs from the ACNAPI Common Service Collection. You can click on the Read more button to view the Postman documentation.

You can further click on the Run in Postman button to open the APIs in Postman.

Get Server Token

Note that you need the Server Token to call any subscribed API.

Navigate to the Authentication tab and copy the Sever Token by clicking on the Copy icon.

Use Server Token to call Register User API

Open the ACNAPI Common Service Collection Postman collection, select the Register endpoint and paste the Server Token into the Server-Token field under the Headers tab

Update the object under Body tab if you wish to and Click Send to call the API

{
    "username":"testing_user",
    "password":"pASSw0-d8",
    "phone":"415-392-0202",
    "gogo": "lala2"
}

You will then be able to see that the api is successfully called.

Check database for the user created

Upon success, if you can navigate to Database > Users (under Browser), you can see that the user is created in the database
